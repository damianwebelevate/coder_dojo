Twitter Bootstrap

Is a responsive framework for use on the web

It uses a system of 12's so your web page is split into 12 columns (infinite number of rows)

For layouts the concept of fixed and fluid remain in the framework however the by 12 split is still present.

Using this framework has many benefits from custom components(slideshows/accordians/menus etc.

Easily create mark-up

Can hide/show content with a simple class declarations such as visible-sm and hidden-xs
