Web page layouts using divs

Fixed Layout

A sample web page and structure for your own study

CSS elements are based on the box model. To see this model in action have a look at the lines around each element on this page.

The black lines represent the div's and the yellow represent the headings/links/paragraphs

The gaps inbetween are either the attribute padding or margin. These attributes are used to provide spacing

For a fixed layout to work you style your page as follows:

In your html page create a div with a class of container

<div class="container"></div>

In your css file:

.container {
  width: 960px;
  margin: 0 auto;
}

The margin attribute centers the container on the page.

The layout you choose can be calculated easily and ensures a rigid and exact design for the web page

Designer has complete control over the layout

Major disadvantage....browsing on a device with a smaller screen resolution makes for a poor user experience.

Minimise your browser window and see what What happens to this layout at smaller screen sizes

This page uses tabindex

Tabindex allows screen readers to use the tab key on the keyboard to navigate through the page. Many web developers are lazy when it comes to this but for accessability and navigation please use where appropriate especially on menus and links. If you look below the tabindex in the next three links is 7,6,5. The links can be accessed via the ordering you set them. Try this yourself and see. Load the web page and use the tab key
