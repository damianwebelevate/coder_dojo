All samples can be found on their site Bootstrap 3.2.0

Open lego_pieces folder and edit carousel.txt

Copy and place inside your index page after the menu section

Clearly comment and seperate for legibility

Refresh your page and you should see your carousel

But its not quite working yet so lets fix that...

Right click the js folder and create a new file.

Name it myfunction.js

Start the file with the following:

$(function(){

end the file with this:

});//end of file

To initialise the carousel we pass the dom reference in jQuery with the dollor sign: $

Write in the text inside the opening and closing function as above, like this:

$('.carousel').carousel();

The bootstrap javascript library extends the jQuery library. What we did here is saying...find all references to an element with the class name carousel and initialise it with the function called carousel. This carousel function in older versions of bootstrap.js used to accept an interval (time value) but this has since changed. Using this sets the carousel to play automatically.

For next week

You will notice that your pages have menu titles with To Create

Create these files yourself.

Link to them

In each of the pages give your body tag an id that you can target with javascript. example: <body id="index">

An id is meant to be unique so please change the name of the id each time

In your myfunction.js file complete the following for each page:

$("#index a:contains('the_name_of_your_menu_title')").parent().addClass("active");

# in javascript or in css represents the id attribute.

In javascript there is a concept of bubbling.

This means essentially that the parent element of an another element can be targeted and we can reference the parent to effect the child

Sounds confusing? If we have a div with a class of someclass the children of this class can be targeted by css. So we can say div.someclass p

By doing this we are selecting the p(paragraph) that only resides within that div element and no other

Similarly we can do the same with a concept called bubbling.

Events bubble up to its closest parent element in the dom.

This line says from the dom $, select the id denoted by the # key, in this case id="index", a link that contains the text index (a:contains('the_name_of_your_menu_title')).

.parent() the parent element which is the ul and add the class of active in the css file.
